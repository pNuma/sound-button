# sound-button

良い選択効果音を探すために作成

## 機能 
* **サウンド再生**: 大きなボタンを押すことで、選択したサウンドを再生
* **モード選択**: 以下の3つのモードを切り替え
    * **固定**: セレクターで選択されているサウンドを再生
    * **5回で変更**: 5回ボタンを押すごとに、自動で次のサウンドに切り替わる
    * **ランダム**: ボタンを押すたびに、ランダムなサウンドが再生

## 実行


## 使い方

1.  セレクターから好きなサウンドを選択
2.  画面下部のボタンで再生モード（固定 / 5回で変更 / ランダム）を選択
3.  中央の大きな再生ボタンを押すと、サウンドが再生される
